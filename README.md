# Adaptive Cyber Threat Hunting Platform

ACTH-P is a modern cyber-defense dashboard that helps analysts detect, score, and investigate threats in real time.
It combines AI-based risk scoring, MITRE-aligned insights, and a beautiful SOC-grade interface to make threat hunting fast, visual, and impactful.
Designed for portfolios, university applications, and cybersecurity interviews  ACTH-P shows serious hands-on security engineering skills.
